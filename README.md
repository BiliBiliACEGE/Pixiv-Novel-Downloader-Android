# Pixiv小说下载器 - Android版

这是一个用于下载Pixiv小说的Android应用程序。

## 功能
- 下载单个Pixiv小说
- 下载整个小说系列
- 支持TXT/HTML/Markdown格式
- 多语言支持(中文/英文/日文)
- 暗黑/明亮主题切换

## PC版本
PC桌面版仓库: [https://github.com/BiliBiliACEGE/Pixiv-Novel-Downloader](https://github.com/BiliBiliACEGE/Pixiv-Novel-Downloader)

## 使用说明
1. 输入Pixiv小说ID或链接
2. 选择下载格式
3. 点击下载按钮

## 构建
使用Android Studio导入项目并构建
